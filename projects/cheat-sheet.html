Looping cheat sheet

While Ruby has more loops to use compared to Javascript, the two they share in common act similarly. The syntax is rather different between the two. I've created this cheat sheet to provide a quick comparison of the syntax between the two.

Ruby Loops:

for used to repeat action a desired amount of times. 
for n in 1...50 do
 puts "hi"
end
Can also be written as: 
50.times do |n|
 puts "hi"
end
while runs while conditional is true 
while i < 100
 puts "hi"
 i += 1
end
until The opposite of while, until runs until the conditional is false 
n = 10
until n < 2
 puts "hi"
 n-=1
end
each each can be used to iterate over data sets 
cars = [sedan, truck, hatchback, convertible]
cars.each { |type| puts type }
also written as 
cars.each do |type|
 puts type
end

Javascript Loops:

while runs until condition is false 
while (n < 23) {
 puts "hi";
 n++;
}
for loops a number of times 
for (var i = 0; i > 50; i++){
 puts "hi";
}
